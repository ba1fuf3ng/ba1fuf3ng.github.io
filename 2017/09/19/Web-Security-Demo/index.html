开放式Web应用程序安全项目（OWASP）公布的最新十大WEB安全漏洞


OWASP TOP 10 2017

WEB安全三境界

  • Tools User 工具小子
    对各种漏洞原理浅浅了解，缺乏Web站点开发与运维经验能力，缺乏对HTTP协议的掌握，只能略读懂（甚至不懂）一种或多种脚本语言。通常只会使用别人开发的攻击测试工具（以Windows GUI为主），以猜测弱口令、自主利用“BB型”漏洞或者使用他人漏洞利用脚本工具为自豪感来源。

  • Scripts Maker 脚本小子
    对各种漏洞原理基本掌握，具备一定的Web站点开发与运维经验能力，基本掌握HTTP协议，能够简单审计至少一种脚本语言。能够按照实际需求开发WEB攻击测试脚本，了解浏览器安全

  • Web Hacker 安全研究员/WEB黑客
    深谙各种漏洞原理，具备丰富的Web站点开发与运维经验能力，熟悉HTTP协议，具备独立审计WEB应用并发现0day漏洞的能力，研究浏览器安全。能够按照需求开发WEB攻击测试框架。

  • Master 大师
    研究WEB安全前沿领域，培养WEB安全人才。
